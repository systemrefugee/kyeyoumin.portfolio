Youmin Portfolio (JSON-driven)
=================================
1) Open the folder in VS Code
2) Run with Live Server (index.html → Open with Live Server)
3) Add your images into the /images folder, then update paths in projects.json
4) To add a new work, append an object to projects.json (title, tags, thumb, images, descs)
